# wasm-js

WebAssembly showcase for node.js
Runs the fibonacci sequence as a wasm module in node.js

language of implementation is c

## Dependencies

clang, npm, node

## Build

```sh
npm run clang:compile
```

## Run

```sh
npm start
```
